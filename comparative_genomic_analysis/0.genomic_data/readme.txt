The "pep" directory contains protein sequences for various species, with filenames in the format of XXX.pep.fasta, where XXX represents the species with three uppercase letters.
The "cds" directory contains cds sequences for various species, with filenames in the format of XXX.cds.fasta, where XXX represents the species with three uppercase letters.

Note: The sequence IDs in the files need to be named in the format of XXX_symbol(or number).
